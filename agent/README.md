# Agent Layer

This folder exposes the minimal public-safe agent layer for the current `Study Superpowers` slice.

## What Is Included

- a root [AGENTS.md](../AGENTS.md) for execution boundaries
- one minimal operator skill
- one public-safe handoff skill for packet ownership transitions
- one public-safe repair-scoping skill for patch-home choice
- one public-safe execution-receipt skill for compact operational close-out
- machine-readable routing through [../task-router.json](../task-router.json)
- public docs that explain scope and architecture

## What Is Not Included

- private maintainer routing
- private runtime logs
- learner-specific project state
- the full private support stack from `study`
