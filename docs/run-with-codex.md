# Run With Codex

Use this repository as a public-safe operator slice, not as the canonical owner of a private workspace.

## Minimal Flow

1. Read `AI_CONTEXT.md`.
2. Read `docs/public-scope.md`.
3. Choose one lane from `task-router.json`.
4. Run `pwsh -NoProfile -File ./tools/Test-All.ps1 -RepoOnly` before and after public-surface edits.
5. Keep maintainer-only working state in the private owner bundle back in `study`, not here.

## Good Fits

- repo-safe validation
- public-boundary checks
- support-slice packaging edits
- AI-facing onboarding repair
- packet-owner handoff into ops-shaped work
- repair-scope and patch-home choice before implementation
- execution-receipt close-out for ops-shaped work
